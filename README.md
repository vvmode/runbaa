# Runbaa Tech — runbaa.tech

Corporate website for Runbaa Tech Pvt. Ltd., built as a static Vue 3 + Tailwind CSS single-page site with Vite.

## Getting started

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

Outputs a static site to `dist/`, ready to deploy to any static host (no server/database required).

## Structure

- `index.html` — Vite entry point
- `src/main.js` — Vue app bootstrap
- `src/App.vue` — root component
- `src/components/` — page sections (Nav, Hero, About, Services, Portfolio, Team, Contact, Footer)
- `src/style.css` — Tailwind CSS entry
